The `CARGO` class contains constants representing different corporate positions and officer roles that appear in BORME (Spanish Official Gazette of the Commercial Registry) filings.

## Overview

CARGO defines 400+ different corporate positions, from basic roles like President and Secretary to specialized positions in committees, boards, and special situations. It includes mappings from Spanish abbreviations to full position names.

## Keyword Mappings

CARGO provides extensive mappings from Spanish abbreviations and variations to full position names:

```python
from bormeparser.cargo import CARGO

# Access all keyword mappings
keywords = CARGO.KEYWORDS

print(len(keywords))  # 600+ variations
```

## Usage Examples

### Accessing Position Constants

```python
from bormeparser.cargo import CARGO

# Access specific positions
print(CARGO.PRESIDENTE)  # "Presidente"
print(CARGO.CONSEJERO_DELEGADO)  # "Consejero delegado"
print(CARGO.DIRECTOR_GENERAL)  # "Director general"
```

### Converting from Abbreviations

```python
from bormeparser.cargo import CARGO

# Convert abbreviated forms to full positions
abbreviation = "Pdte."
if abbreviation in CARGO._keywords:
    position = CARGO._keywords[abbreviation]
    print(f"{abbreviation} = {position}")
    # Output: Pdte. = Presidente

# Using the from_string method
try:
    position = CARGO.from_string("Cons.Delegado")
    print(position)  # "Consejero delegado"
except BormeInvalidCargoException as e:
    print(f"Unknown position: {e}")
```

### Filtering by Position Type

```python
from bormeparser import Borme
from bormeparser.cargo import CARGO
from bormeparser.acto import ACTO

# Parse BORME file
borme = Borme.from_json('borme.json')

# Find all CEOs appointed
for company in borme.get_companies():
    acts = company.get_acts()
    
    if ACTO.NOMBRAMIENTOS in acts:
        appointments = acts[ACTO.NOMBRAMIENTOS]
        
        for person, cargo in appointments:
            if cargo == CARGO.CONSEJERO_DELEGADO:
                print(f"{person} appointed as CEO of {company.name}")
            elif cargo == CARGO.PRESIDENTE:
                print(f"{person} appointed as President of {company.name}")
```

### Identifying Executive Positions

```python
from bormeparser.cargo import CARGO

# Define executive positions
executive_positions = [
    CARGO.PRESIDENTE,
    CARGO.CONSEJERO_DELEGADO,
    CARGO.DIRECTOR_GENERAL,
    CARGO.PRESIDENTE_EJECUTIVO
]

# Define board positions
board_positions = [
    CARGO.PRESIDENTE,
    CARGO.VICEPRESIDENTE,
    CARGO.CONSEJERO,
    CARGO.SECRETARIO,
    CARGO.VICESECRETARIO
]

# Check if a position is executive
def is_executive(position):
    return position in executive_positions

# Check if a position is on the board
def is_board_member(position):
    return position in board_positions
```

### Complete Example

```python
from bormeparser import Borme
from bormeparser.cargo import CARGO
from bormeparser.acto import ACTO

# Parse BORME
borme = Borme.from_json('borme-2024-01-15.json')

# Analyze corporate governance changes
for company in borme.get_companies():
    print(f"\n{company.name}")
    print("=" * len(company.name))
    
    acts = company.get_acts()
    
    # Appointments
    if ACTO.NOMBRAMIENTOS in acts:
        print("\nNew appointments:")
        for person, cargo in acts[ACTO.NOMBRAMIENTOS]:
            print(f"  {person} - {cargo}")
    
    # Resignations
    if ACTO.CESES_DIMISIONES in acts:
        print("\nResignations:")
        for person, cargo in acts[ACTO.CESES_DIMISIONES]:
            print(f"  {person} - {cargo}")
    
    # Revocations
    if ACTO.REVOCACIONES in acts:
        print("\nRevocations:")
        for person, cargo in acts[ACTO.REVOCACIONES]:
            print(f"  {person} - {cargo}")
```

## Position Hierarchy

Typical Spanish corporate governance structure:

```
Junta General de Accionistas (General Shareholders Meeting)
  |
  +-- Consejo de Administración (Board of Directors)
      |
      +-- Presidente (Chair)
      +-- Vicepresidente (Vice Chair)
      +-- Consejero Delegado (CEO/Managing Director)
      +-- Consejeros (Directors)
      +-- Secretario (Secretary)
      |
      +-- Comité de Auditoría (Audit Committee)
      +-- Comité de Nombramientos y Retribuciones (Appointments & Remuneration)
      +-- Comisión Ejecutiva (Executive Committee)
      |
      +-- Dirección General (Management)
          |
          +-- Director General (General Manager)
          +-- Director Financiero (CFO)
          +-- Director Comercial (Commercial Director)
          +-- etc.
```
